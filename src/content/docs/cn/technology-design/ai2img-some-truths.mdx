**摘要**： AI绘画已经事实上应用于各个行业中，本文从设计师的实际观察和应用出发，得出一些观点。

## 一、AI绘画根本不是一场创意的盛宴

昨天我在跟随一个设计课程，需要制作一张宣传单，尽管课程要求仅仅是草图，我依旧准备制作一个完全的插画版本。
这是我的草图：
![work smarter not harder - sketch by libregd](/images/AI2img/rework-flyer-sketch.png)

*work smater not harder* 基本已经是一句被做烂了的题材，如果简单的检索就可以发现下面的图片：
![work smarter not harder](/images/AI2img/work-smarter.png)

这么看我的草图还不错，确实有自己的创意在里面。然后我需要面对的就是执行：细化细节，调整风格，让它看起来漂亮。—— 说实话，我最讨厌的就是细化细节，这一点和很多设计师不一样，大部分设计师精益求精，注重雕刻精致的细节，提高整体的视觉质感，绝大多数画画的人在过程中会收获非常明显的心流体验。—— 而我很少在绘画中体会到。

突然一个念头，我想试试看AI绘画能否做做看呢？ 虽然我之前已经用AI来尝试了字体设计，得出了AI绘画并非“学习”绘画，更多是融合的[观察结论](https://x.com/libregd60294/status/1976818864299425899)。简要的复述我的观点：
1. 即便是专业的设计师一天也不可能将全世界新出现的图片看过，更别说记住。
2. 字体设计不需要任何的专业素养也能从中品味到创意和美——这大大降低了讨论AI的背景。
3. 在这次字体设计测试中，AI从专业技法到创意都是0.

但这次的草图明显更贴合大部分设计师从业者在实际工作中的应用情景，能否更明确的让这个结论更普遍性呢？—— 并非拿着锤子找钉子，只是好奇的尝试。

## 二、过程反应事实

**风格确定**：
1. 由于是宣传单，那么配色上最好鲜艳明亮一些，撞色可以提高醒目的程度。
2. 质感不仅在印刷工艺上相对好实现，成本可控，也更容易一次素材二次利用：适配线上传播。

**希望AI帮我做什么?**
1. 风格转换，因为核心的创意概念，元素排版已经完成了，需要AI帮我做风格的转化和细节完成。
2. 最好能有透明的底，这样方便处理排版。

**三个关于AI出图的事实**
```
promptA:转换风格，更多的质感纹理，现代扁平高级插画质感。
```
![mountain](/images/AI2img/case0.png)

**事实1**：AI并不理解画面里有什么，它只是提取了轮廓，根据概率判断出占比最大的部分是什么。

```
promptA:转换风格，更多的质感纹理，现代扁平高级插画质感。
promptB:更多的质感纹理，现代扁平高级插画质感。 绘制一张透明底的 gondola （登山的缆车）
```
![gondola](/images/AI2img/case1.png)
**事实2**：如果AI是真的会画画，比如prompt B， 那么prompt A的结果不会如此糟糕。
```
promptA:转换风格，更多的质感纹理，现代扁平高级插画质感。
promptB:更多的质感纹理，现代扁平高级插画质感。 绘制一张透明底的 拄着登山杖正在爬山的人
```
![climber](/images/AI2img/case2.png)
**事实3**：比起绘画，更像从别人的作品里撕下了一块拼图。

**最终结果**
![rework-flyer-result](/images/AI2img/result.png)

说实话完成度相当不错，但显而易见的事实是，如果我用了别人创作的素材，通常情况下要么寻找免费的素材库，要么付费购买授权。
如果你知道AI当了小偷，你还能心安理得的在商业场景下侵犯别人的收益吗？

## 三、版权，设计意图，效率陷阱

### 以前的免费靠爱，现在的免费靠AI帮忙偷

草图的绘制大概花费了1h，从概念到矢量的完成，通常后续的细化需要花费3-8h 取决于结果的细致程度或者**可能出现的别的版本**。而AI的版本，仅仅在草图的基础上，花费了10-20分钟，就获取到了AI提供的“素材”。

说实话，作为自学入行的设计师，其实过去很长一段时间都有点分不清原创，借鉴，抄袭 之间的区别，甚至犯过一些错。

AI出现后，好像世界的底线变得更低了。

### **AI真的能替代设计师吗？**

我觉得能O(∩_∩)O，虽然我并不算设计非常优秀的人，但非常擅长快速学习做出外行眼里及格的东西，这种特质让我对理解需求非常敏感，并且策略上偏重抓大放小，而非精益求精。

靠设计吃饭，很多时候和靠文字吃饭性质非常类似：他们多少给人“**空手套白狼**”的感觉。

作文谁都会写，但你会为什么样的文字花钱？—— 这个问题拿来问设计也是同样的，无非是文字的鉴赏训练人人都经历过，而艺术/视觉的品味培养，则只能靠个人境遇了。

### **效率陷阱**

当工作流极度高效，创作被量化，设计师会思维趋向固定化，比如我，做一张练习用的宣传单，并非首要考虑尝试新的视觉风格，而是侧重最稳妥的方案。如果再加上对效率的渴求，那么在过程中最宝贵的“突然想到”，“灵光一现”，“我想试试” 会越来越少。—— 这些不一定会让甲方更好，但多少会让这个世界更美好，会让设计师这个人更美好。

**如果所有人都能快速产出“好看”的作品，设计师何去何从**

视觉风格也并不稀缺，在AI时代，可以快速的测试风格。精致的细节和完成度也不稀缺，毕竟AI还挺会偷的。
