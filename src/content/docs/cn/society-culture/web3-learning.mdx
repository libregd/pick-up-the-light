## 0. 我与 Web3 的相遇
最近在考虑混入web3的设计师，出于寻求职业机会&记录反应时代的角度，突击进行一些扫盲式补习。
毕竟 远程+新兴机会 合适的选项并不多，假装web3/blockchain 不重要挺难的。

### 你对Web3 / 区块链的第一印象是什么？是“技术革命”“骗局”，还是“未来趋势”？
最近才开始认真了解，但很早就断断续续听过一些基础概念和新闻。这个早可以追溯到2009-2016.

我一方面觉得技术革命+未来趋势， 一方面感觉也挺骗局的。这并不矛盾。

频频爆出的一夜暴富，十万美元大关 —— 让显得没有参与这场游戏的人都有点傻了：如果是骗局怎么会这么多人甘之如饴，而且还多是以高学历高科技背景为主的人群为自己的认知买单？最重要的是，强调的预期蓝图很动人：完全自主的管理自己的财产，不受政府随意发行货币的影响 —— 普通人会直接从这一颠覆性的变革收益。

如果真是个好游戏，为什么看不到到底提供了什么可交易的“商品”？
我是说，价值不可能凭空出现，比如一位艺术家，花费时间精力创作了画作，然后被人买下，多高的价钱都是合理的。（尽管“烂在手里”这个形容的存在说明收藏家这个身份还是门槛太低了） —— 但区块链呢，你告诉我有人通过下载安装程序，耗费电力，电脑不关机运算，然后获得了一笔巨款，传销都比它合理好吧，传销还得发展人头拉人赚个入会费呢！

### 到目前为止，你的理解与最初相比有什么变化？

变化挺大的。

区块链刚出来的时候除了石破天惊还有群魔乱舞，很多时候往往是群魔乱舞比较多，毕竟在断断续续看到的新闻里，区块链是一种技术，并不是一种稀缺品。任何人都可以按着自己的心意发行自己的货币。—— 极致的自由往往伴随着极致的混乱。

不过这种混乱总会慢慢建立自己的秩序，毕竟区块链技术没问题，理念没问题。—— 那么真正的创造者为什么要把好好的土壤让给为非作歹的混蛋呢？

所以现在的区块链/web3 生态逐渐走向了与现实更多的链接，越来越多的人投身在这个行业来做基础建设，就想当年web2一样，想要蓬勃的互联网，即便是中心化的，总要有人迭代优化 各种技术/规范/法规 的问题吧？

以前听到的都是各种五花八门的币，现在除了这些币，还能看到切实有币的应用场景和生态。只有有人真实的在里面交易通过劳动提供的东西 实物/虚拟产品/服务...... —— 那就是成立的。

## 1. 区块链的核心逻辑

### 你认为区块链解决的“根本问题”是什么？

用通俗的话来说：记账本机制让干坏事的成本变高了。—— 以前需要靠中心化的暴力/权威机构来维护 ，现在则变成我为人人，人人为我。—— 每个节点都在记账验证存证，别人也在为你做同样的事。

用正式一点的说法：在没有中心化权威的情况下，如何建立多方之间的“可验证信任”。它用技术手段替代制度强制，让数据和行为在系统内自动守规。

并不是说区块链不会带来新的困难，而是这些“新困难” 在没有区块链的时候一样存在啊......

### 它与互联网（Web2）的结构有何不同？
体感上对普通用户区别不大，其实还是在app/website 之间打转，毕竟物理技术迭代没那么快，web3比起web2更多是玩法规则的转变。

以前最起码你得靠邮箱注册各种平台，个别的平台还需要身份证/手机号码 之类的。—— 也就是说，你证明你是你，你还得需要依靠“权威/可信”的平台为你作保。而在web3，你只需要钱包地址，而这个钱包地址只要你不忘记唯一的一串密码，那谁也偷不走。

即便是免费用户，也在为平台贡献日活——看广告，凑热度。如果你再努力一点当了KOL，带货不走平台的渠道，平台分不到钱可是要惩罚你的哦。

   > 如果你不带货，只是想把粉丝转移到别的地方，也会惩罚你的哦。—— 来了就是打工人，还想白嫖？

而在web3，其实在变现上虽然并没有特别的优势，因为对消费者来说核心不在于我使用了什么币，而在于我付出什么交换什么。——但web3依旧提供了更多的可能性：你的规则写在合约里，人人可查阅，内容可以存储在区块链，粉丝可以迁移，跨平台组织活动... 

这些都让人在除了带货以外，可以有更多的创作变现的可能性，而非单一的依靠上交保护费获得平台流量倾斜机制。另外不再怕封号。


### 去中心化在技术上和社会意义上分别意味着什么？

技术上我感觉更安全了，虽然数据上有些冗余，不过这种冗余比真实世界的成本低多了，一个极低的成本换取更大的益处是值得的。虽然也要提防多个节点被一口气篡改就是了......

社会意义上会进一步让个体主权更明确，其实对人的要求也提高了，比如web2阶段没几个人会一条条的看“软件安装权益声明”，web3人人化身阅读理解大师，毕竟自己的资产自己负责，合约不看明白被坑了自己怪自己...—— 从这个角度来看，具备一定程度的计算机安全知识的黑客还挺容易在web3做自媒体的，面向大众那种。

小红书刚出来我还嘲笑，怎么会有人这么热衷于看说明书啊...... —— 直到查阅一些自己完全没头绪也没耐心的内容，确实挺香的。

另外创作变现更为合理，以我熟悉的网文为例子，除了极少数头部，大部分网文创作者说是给平台做日活零工并不为过，即便如起点这样的第一平台，也出过“买断风波”这样的恶性事件。更不如说番茄了，不签约不推荐，签约了版权，改编权都是平台的。甚至还允许AI随意训练。—— 明明作者选择了阅读完全免费的高难度模式，反而获得了更可怕的未来。

web3 会更公平吗？—— 这世界从来不太公平哈哈哈，但你至少可以多一种选择，用脚投票，公开透明的参与DAO社区治理。

### 你怎么看待“信任”的重新定义（code vs institution）？

我觉得蛮好的，翻旧账打官司还得拿出各种证明材料呢，如果链上过程直接就是证明材料，那很多事情不用掰扯，事实会说话。

说了这么多年的工作留痕，这不就最合适区块链技术上链。—— 搞笑的是，老中就不怎么着急引入了。明明没有比中国人更合适开展区块链应用的地方了：又不怎么在意隐私（主要是老百姓朴素的“没啥好看的”想法深入人心）， 实名都上网了，实名区块链钱包打通乱七八糟的平台很难吗？实名举报都有了，也不能白举报吧？上个链也很方便追查啊.....——充分说明规矩只是管理手段，而非社会公约。

## 2. CEX / DEX / DeFi 的机制与差异

### 在你看来，CEX（中心化交易所）存在的合理性是什么？

我是感觉约等于金融机构但运作的不是法币，而是加密货币，并提供加密货币-法币 之间的转换。
目前规模是DEX的数十倍，是很多人的首选，毕竟买卖货币不像买菜，在哪里买都一样，新鲜干净那是一眼可查，不用讲究门头口碑。所以头部CEX有绝对优势，主要是多年经营的信誉积累，CEX主要的获利手段是交易手续费。

另外CEX的用户很多都本身对交易是有经验的，所以体感上还挺像股票交易所 but 7*24h 实时买卖，非常刺激...


### 它的问题又在哪里？

最主要的还是有点中心化了，另外资产托管给CEX平台的钱包系统（等于充值平台专属钱包）后，还是有一定的风险面临跑路，黑客攻击。

如果你又想选声誉大的机构，又想不暴露真实身份 —— 那么这些平台会拒绝你，因为本身还是和法币高度关联，玩加密货币但你身份不加密。

你能买卖的是可以参与买卖的，对大多数人来说并不能自己造血变成LP。

### DEX（去中心化交易所）的核心价值是什么？

1. 你自己的钱包直接参与交易，不需要转进某些CEX平台。
2. 你可以自己作为LP投一笔资金设定一套运行规则，让别人以你的规则来玩，具体交易什么取决于你的生态开展的业务。
3. 既然规则是你设定的，你还能自己从交易手续费中收取你应得的部分。
4. 总的来说，DEX鼓励激活最小单元的组织者，提供了一套基础建设，给那些愿意尝试把传统线上交易一步步迁移到web3来的人。最典型的就是网红/KOL/IP经济， 本身这些交易大概率是只买入不卖出，卖出也只存在认可价值的圈子玩家，比如labubu，只能卖给认可labubu价值的群体。即然真金白银可以买，那么专属的某种代币为什么不可以？

### 你能否举例说明一次典型的 DEX 交易流程？

假设一个场景：有一个网红Ken，发行了专属代币 I'am ken, 初始定价比例和美元是1：1， 
1. 专属代币持有2000个1周以上，可以直接附赠专属的Ken表情包NFT一套，
2. 专属代币持有20000个6个月，可以用8折的价格兑换6个月后的演唱会的VIP嘉宾座椅（限量1个），并且解锁演唱会会后独占纪念数字版内容。
3. 可以代币持有的数量可以按比例分配下一次Ken选题的投票权重占比。ken以外的人最多可以持有49% 
4. ken币一共只发行200w枚，作为全网粉丝2000w的网红，具备稀缺属性，并且他正值创作的旺盛期，相当多的限量周边只有代币才可以购买。

作为粉丝我能做的事：
1.  连接钱包，使用USDC 兑换 Ken，熬大夜就像抢演唱会门票一样，掐着点换取了2000个
2.  虽然大家都挺活跃导致滑点略大还拥挤gas费有点高，但整个过程交易可查。
3.  随后起起伏伏，ken出轨导致ken币价格暴跌，我趁机低价买入。
4.  然后ken浪子回头，价格又起来了，毕竟这个家伙还是该死的有魅力。（怀疑出轨是演的的程度）
5.  随后用代币购买了VIP座位，然后这是可以出售的。净赚了差不多一倍价格，演唱会还是太抢手了。
6.  当然也是用的ken币，我和那位粉丝的交易手续费0.3% 给到了Ken，毕竟没有他做这个资金池，这场交易就不存在。
7.  ken币虽然暂时不能出门买菜，但可以用ken币转成USDC，甚至法币。

总结一下：
1. 代币数量可代表忠诚度、参与度
2. 可空投、质押、奖励早期支持者
3. 可用于投票、提案、参与创作方向
4. 可交易、可涨跌，形成“粉丝经济”

### 从用户体验、信任模型、风险控制三个维度比较 CEX 和 DEX。

这个问题问AI可以得到非常丰富的表格，所以没贴......

**用户体验对比**

CEX是版本3.0的理财APP，老道又丝滑 。DEX是野蛮生产的一团乱麻的版本0.1，生机勃勃...

**信任模型对比**

CEX是平台还是你爹，DEX是自己给自己当爹，或者任何人都可以当别人的爹.(有爹不一定是坏事...)


**风险控制对比**

CEX延续了传统金融的精神，不需要从头发明规范,但CEX即怕贼偷也怕贼惦记，资产被盗事件蛮多的。

DEX更容易被人投机，Rug Pull对生态的信任感破坏严重。 除非这个LP你真的很认可，不然你肯定是个老赌徒了。

**判断一个 DEX 代币是否“不可取代”的 4 个问题：**
1. 是否绑定协议治理？
2. 是否参与收入分配？
3. 是否承担激励或安全角色？
4. 是否具备链上合约强绑定？

**如何避免 DEX 做市（Rug Pull）风险？**

1. 选择成熟协议
2. 查看流动性池的历史变动，是否有频繁撤池行为
3. 避免参与新币或匿名项目的做市，尤其是未审计合约
4. 使用 DEX 聚合器或审查工具
 
### DeFi 在这两者的框架中扮演了什么角色？它到底“去中心”到什么程度？
以下这部分整理自AI

**🧩DeFi 在 DEX 框架中的角色：原生基础设施**

1. **可组合的金融乐高**： DeFi 是 DEX 的底层逻辑和生态组成部分
2. **无需许可的金融服务**：所有功能（交易、借贷、衍生品、保险）都通过智能合约模块组合实现 
3. **资产自托管 + 自动执行 + 透明治理** ：用户即参与者、治理者、流动性提供者，拥有资产主权    
4. 面临合约风险、MEV、无常损失、治理攻击等链上原生风险    
---

**🏦 DeFi 在 CEX 框架中的角色：产品补充与流动性来源**

1. **收益型产品** ：DeFi 是 CEX 的“外部资产池”或“产品补充”，并非核心架构
2. **链上资产交易对** ： 通过链上桥接、API 接入、DeFi 产品包装，用户通过中心化界面间接参与 DeFi，资产仍托管于平台 
3. **平台竞争力与用户粘性**：平台承担合约风险，用户承担平台信用风险    

---

**✅ 一句话总结**：

> **在 DEX 中，DeFi 是“操作系统”；在 CEX 中，DeFi 是“插件”。**  
> 前者强调用户主权与协议自治，后者强调用户体验与平台整合。
>

我感觉DeFi的去中心化程度蛮高的，但是技术门槛并不低，或者说想把DeFi 落实成web3的基建，是真的有点基建的味道：雄关漫道真如铁，而今迈步从头越。

### 你认为 DeFi 的“创新”是技术的创新，还是金融逻辑的翻新？

更多是技术上的创新，DEX建设的是协议生态，做的是做大蛋糕的建设 ，而Defi是帮助别人做大蛋糕的基础建设。
金融的底色是高效促成交易流动，主要还是靠抽手续费过日子。这一点Defi并没啥不同....

## 3. 理想与现实的张力

### 你觉得“去中心化”是现实可行的理想吗？

可行。

1. 过去几年我已经很久没有在中心化的公开社交平台发表短文了，因为被封号， 我转向了去中心化的社交媒体。可以定期导出自己的数据，并建立更为安全的社交网络。

2. 我已经很久没用过现金了，换句话说，只要能够达到交易目的，其实普通人并不在意用什么购买。疫情期间还以物易物呢，比如香烟换可乐。 

3. 变革肯定是从原本的重web2的资产优先转移到web3，毕竟web2对现实世界的侵蚀也花了一些年头，也并没有完全吞掉，直接推广web3不如扒在web2的身上吸血，一次吸两个。

### Web3 社区的理想主义与实际的投机行为之间，有哪些结构性的矛盾？

1. 说起来是人人可参与，问题早期代币早就被别人拿在手里啦~ 代币一方面是治理凭证，一方面是投机工具。更多的时候是投机工具目前来看，毕竟时至今日还有很多人没听过比特币的名字，如果信息一开始就不对等，那么价值可能并非依靠流动，而是收割。——从这一点来看DEX确实很有魄力，通过从新的劳动价值中定义价值。
2. 人人可参与治理还是太难了，如果不承认别人在一些事情上比自己更擅长，那么世界上美女那么多，为什么没多我一个？—— 尽管我更偏好精英叙事，但目前web3 覆盖的人群还是太少了，这又是一个自古以来的现实：精英如果为大众做牛做马，未必有啥好果子等着你，而利用大众的“愚昧”和善良作恶的精英，大众又实在见过太多......—— 信息优势不仅仅是链上贵族，更意味着更多方面的绝对优势 —— 这种事实让人近而远之。
3. 相当比例的web3 社区的空投，有点像web2时代那些P2P拉新活动。你都不知道存在的意义是什么，但偏偏如火如荼......—— 各种新币排行榜也挺有必要的，某种程度比虚假的社交媒体平台的粉丝数更真实，毕竟要真金白银的支持真材实料的创作者。并非所有的人都喜欢也偏好聪明钱，对更广大的群众而言，如果web3真的有什么值得存在的，必然不是紧赶慢赶的空投套现。（别说大众了，我稍微懂一点的普通人我都不愿意花心思精力在薅羊毛上 ）
4. 挖矿作为一种工作量证明，似乎被视作门槛最低的参与web3社区的办法，但这种办法也往往被少数“懂的人”套利，普通人依旧边缘。—— 我的意思是，普通人明明可以靠自己作为人的存在来创造交换，为什么要靠懂挖矿技术？ 不懂难道不能参与web3 吗？
5. DAO是社区自治的载体，但实际投票慢慢变成一种谁币多谁说了算的中心化游戏，即然代币的未来是替代掉法币的一部分职能，显然现实社会中优秀的社区自治也并不全靠钱说了算啊...
### Token 经济设计在激励创新与制造泡沫之间的界线在哪里？

Token 经济的界线在于：是否能将激励行为转化为真实价值，而不是短期套利。

### 你是否认为 Web3 在今天仍处于“早期互联网”的阶段？为什么？

是的，整体叙事还是比较精英的，并不是说精英化不好的意思，只是早期互联网，有条件有机会上网的也整体是相当小的一部分群体。

### 对你来说，Web3 的吸引力更多来自“技术信仰”还是“权力重构”的可能？

主要是权力重构，虽然这个重构是基于对技术的信任才能建立起来吧......但感觉没有暴力机构强制推行，web3不太可能大面积铺开，毕竟网可以不上，买菜做饭超市是要去的，制作衣服的棉花是地里长的，交的电费水费网费还是只收法定货币......

不过web3 本身也没想着过多干扰就是啦，它本身更像数字空间部分的重构。能从web2中拿回属于自己的东西，这一点已经足够了。
